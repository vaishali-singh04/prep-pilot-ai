# 🚀 Prep Pilot AI

AI-powered exam preparation platform leveraging Google Gemini AI to generate intelligent study materials, summarize notes, create quick revision content, and export study notes as PDFs. Built with the MERN Stack.

---

## ✨ Features

- 🤖 AI-powered Notes Generation
- 📝 AI Notes Summarization
- ⚡ 5-Minute Quick Revision
- 📄 PDF Download Support
- 🔐 User Authentication
- 🧠 Google Gemini AI Integration
- 📱 Responsive UI
- 🔒 Secure REST APIs

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### AI
- Google Gemini API

### Authentication
- JWT

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/vaishali-singh04/prep-pilot-ai.git
```

### Install dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### Run

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server directory.

```
MONGODB_URI=
JWT_SECRET=
GEMINI_API_KEY=
```
